# Mazen Magdy - Portfolio

A professional portfolio website showcasing my work as a Full Stack Developer.

## Features

- **Clean Design**: Modern, minimal interface focused on content
- **Responsive**: Optimized for all devices
- **Fast**: Built with Vite for optimal performance
- **Professional**: Designed to impress clients and showcase expertise

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Structure

- `/src/pages` - Main pages (Home, Work, Contact)
- `/src/components` - Reusable components
- `/src/contexts` - React contexts for state management

## Customization

Update the following to personalize:
- Contact information in [Footer.jsx](0-Me/src/components/Footer.jsx)
- Projects in [Work.jsx](0-Me/src/pages/Work.jsx)
- About content in [Home.jsx](0-Me/src/pages/Home.jsx)
- Skills and expertise in [Home.jsx](0-Me/src/pages/Home.jsx)
